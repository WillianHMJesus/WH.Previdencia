# WH.Previdencia
Projeto fictício que simula uma API do portal de previdência privada.  

## Tecnologias Implementadas:
- .NET Core v3.1
- ASP.NET Core
- ASP.NET WebApi Core with JWT Bearer Authentication
- ASP.NET Identity Core
- Dapper
- .NET Core Native DI
- Swagger

## Arquiteturas:
- Arquitetura Hexagonal
- TDD
- Teste Unidade
- Teste Integração
- SOLID
- Clean Code
- Design Pattern
- Domain Notification
- Repository
